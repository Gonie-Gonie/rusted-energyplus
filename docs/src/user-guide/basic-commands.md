# Basic Commands

Common commands:

```powershell
cargo run -p ep_cli -- --version
cargo run -p ep_cli -- oracle-info
cargo run -p ep_cli -- model inspect <input.epJSON>
cargo run -p ep_cli -- model compile <input.epJSON>
cargo run -p ep_cli -- model plan <input.epJSON>
```

Script classes and release semantics are documented in
`operations/script-index.md`.
