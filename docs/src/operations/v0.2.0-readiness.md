## Verification Command

```powershell

```

This checks v0.1 setup artifacts, runs the Rust/docs/source gates, and verifies
the repo-owned fixture:

```text

  version: 26.1
  object_types: 5
  objects: 5
```

It also verifies the oracle smoke epJSON generated from `1ZoneUncontrolled.idf`:

```text

  version: 26.1
  object_types: 28

```

## Boundary

v0.2.0 is inspection only. It does not validate schema semantics, apply
defaults, convert units, resolve references, build graphs, or simulate. Unknown
object types are intentionally preserved instead of rejected.
