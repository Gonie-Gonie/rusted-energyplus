# v0.5.0 Readiness

Status: ready when `.\scripts\dev.cmd v0.5-verify` passes.

## Release Definition

v0.5.0 locks the first geometry and internal-variable evidence surface:

```text
Zone geometry + surface geometry + construction/material inputs + nominal internal gains
```

Supported comparison commands:

```powershell
eplus-rs compare geometry <input.epJSON> <eplusout.eio>
eplus-rs compare surface-geometry <input.epJSON> <eplusout.eio>
eplus-rs compare construction-materials <input.epJSON> <eplusout.eio>
eplus-rs compare internal-gains <input.epJSON> <eplusout.eio>
eplus-rs compare internal-convective-gain <input.epJSON> <eplusout.eso>
```

## Completion Checklist

| Requirement | Evidence |
|---|---|
| zone geometry EIO comparison | `compare-geometry-smoke.ps1` compares `Zone Information` rows |
| surface geometry EIO comparison | `compare-surface-geometry-smoke.ps1` compares `HeatTransfer Surface` class, area, azimuth, and tilt |
| construction/material input comparison | `compare-construction-materials-smoke.ps1` compares `Construction CTF` and `Material CTF Summary` rows |
| nominal internal-gain comparison | `compare-internal-gains-smoke.ps1` compares `OtherEquipment Internal Gains Nominal` rows |
| convective internal-gain trace | `compare-internal-convective-gain-smoke.ps1` compares the first ESO convective-gain series |
| manifest evidence | `surface_geometry_001` declares static surface geometry output requests |
| regression wiring | `compare-regression` includes `surface-geometry` in the input-static smoke suite |
| false-claim boundary | manifests use `comparison_class = "smoke"` and `conformance_claim = false` |

## Verification Command

```powershell
.\scripts\dev.cmd v0.5-verify
```

## Boundary

This milestone does not support fenestration/shading geometry comparisons,
non-world coordinate-system variants, full `GlobalGeometryRules` parity,
surface heat-transfer conformance, zone air temperature conformance, HVAC, or
plant simulation. The evidence remains smoke-level and input/static until
declared tolerances, Rust result artifacts, and blocking gates are added.
