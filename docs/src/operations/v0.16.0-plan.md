existing `plant_loop_diagnostic_001` PlantLoadProfile fixture.

conformance claim.

  for the same loop and equipment rows requested by `plant_loop_diagnostic_001`.

  `algorithm_parity: false`, source-map, timestamp, warmup, and sizing-period

  count, result series count, nonzero projected values, and non-claim markers.
