# v0.7.0 Readiness

Status: in progress.

## Release Definition

v0.7.0 makes comparison runs inspectable and reproducible:

```text
compare smoke suite -> trace.json + compare-summary.json + compare-report.md
```

Supported command:

```powershell
.\scripts\compare-regression.cmd
```

## Current Suite

- `schedule-value`: Rust constant schedule trace against EnergyPlus ESO
- `weather-drybulb`: Rust EPW dry-bulb reader against EnergyPlus ESO
- `zone-temperature`: Rust first-zone `ResultStore` against EnergyPlus ESO

## Artifacts

The regression driver writes these files under
`.runtime\compare-regression\26.1.0`:

- `trace.json`: case start/finish events with status and duration
- `compare-summary.json`: suite status, case outcomes, and first divergence
- `compare-report.md`: markdown summary suitable for release notes
- `profile-summary.json`: profile skeleton with wall-clock durations

## Verification Command

```powershell
.\scripts\compare-regression.cmd
.\scripts\check.cmd
```

## Boundary

The current divergence detector reports the first failed smoke case and captures
its output tail. Numeric first-sample divergence, meter comparison, full output
variable registry comparison, and performance counters remain future v0.7+
work.
