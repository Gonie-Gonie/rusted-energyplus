# v0.1.0 Readiness

Status: ready when `.\scripts\v0.1-verify.cmd` passes.

## Release Definition

v0.1.0 is the first public runnable release. It must build an `eplus-rs.exe`
artifact and run user-visible commands against real epJSON input.

Supported commands in this release:

```powershell
eplus-rs model inspect <input.epJSON>
eplus-rs model compile <input.epJSON>
```

`model inspect` is the primary supported workflow. `model compile` is a typed
model preview verified against repo-owned positive/negative fixtures and the
oracle-generated smoke epJSON.

## Boundary

v0.1.0 is not a simulator and does not validate full schema semantics, build
graphs, generate execution plans, or simulate. RawModel inspection intentionally
preserves unknown object types instead of rejecting them. Typed compile is a
preview for the first seed object families and is not yet the complete supported
object contract.
