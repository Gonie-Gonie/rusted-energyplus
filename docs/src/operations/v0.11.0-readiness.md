## Promotion Requirements

A node output can move from diagnostic-only to conformance only after all of
these exist:

- a conformance manifest with explicit tolerances
- Rust result artifacts for matching keys, variables, and frequencies
- timestamp and warmup handling notes
- first-divergence reporting
- blocking release gate
- source-function map for the EnergyPlus node state registration and update path

Until then, node temperature, humidity ratio, mass flow, and setpoint variables
remain diagnostic-only or baseline-only evidence.
