# v0.5.0 Readiness

Status: smoke-ready when `.\scripts\dev.cmd v0.5-verify` passes.

## Release Definition

v0.5.0 locks the first static geometry, construction, and internal-gain smoke
evidence surface:

```text
Zone geometry + surface geometry + construction/material inputs + nominal internal gains
```

Supported comparison commands:

```powershell
eplus-rs compare geometry <input.epJSON> <eplusout.eio>
eplus-rs compare surface-geometry <input.epJSON> <eplusout.eio>
eplus-rs compare construction-materials <input.epJSON> <eplusout.eio>
eplus-rs compare internal-gains <input.epJSON> <eplusout.eio>
eplus-rs compare internal-convective-gain <input.epJSON> <eplusout.eso>
```

## Locked Variables

| Domain | Oracle source | Locked fields |
|---|---|---|
| zone geometry | EIO `Zone Information` | surface count, floor area, volume, exterior gross wall area |
| surface geometry | EIO `HeatTransfer Surface` | class, net area, gross area, azimuth, tilt |
| construction/material | EIO `Construction CTF`; `Material CTF Summary` | construction count, outside material identity, layer count, thermal conductance, material thickness, conductivity, density, specific heat, thermal resistance |
| internal gains | EIO `OtherEquipment Internal Gains Nominal`; ESO `Zone Total Internal Convective Heating Rate` | zone floor area, equipment level, equipment per floor area, latent/radiant/lost/convected fractions, first hourly convective trace |

## EIO Trust Boundary

The EIO parser is an oracle extraction layer. v0.5 trusts EnergyPlus EIO rows
only as selected static summaries for the dedicated smoke fixtures. It does not
turn EIO availability, CTF summaries, or internal-variable rows into a surface
heat-transfer, zone heat-balance, solar, HVAC, or whole-simulation conformance
claim.

## Verification Command

```powershell
.\scripts\dev.cmd v0.5-verify
```

## Boundary

The dedicated v0.5 fixtures use
`GlobalGeometryRules,UpperLeftCorner,CounterClockWise,World`. This milestone
does not support alternate `GlobalGeometryRules` combinations, non-world or
relative coordinate-system variants, zone origin/rotation variants, vertex
ordering variants, degenerate-surface diagnostics, fenestration/shading
geometry comparisons, solar/radiation comparisons, surface heat-transfer
conformance, zone air temperature conformance, HVAC, or plant simulation. The
evidence remains smoke-level and input/static until declared tolerances, Rust
result artifacts, compare reports, and blocking gates are added.
