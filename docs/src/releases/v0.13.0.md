## Scope

Included:

- typed `PlantLoop`, `Branch`, `BranchList`, `Connector:Splitter`,
  `Connector:Mixer`, and `ConnectorList` intake.
- typed identity records for `Pump:ConstantSpeed`, `Boiler:HotWater`, and
  `Chiller:Electric:EIR`.
- plant node registration for loop sides, branch components, and first
  equipment identities.
- graph edges from plant loops to branch lists, branch lists to branches,
  connector lists to connectors, and branches to component records.
- CLI summaries for plant typed counts and plant graph edge counts.
- v0.13 release verification that checks source files, docs, Rust markers,
  smoke output, tests, false-claim boundaries, evidence packaging, and package
  contents.
- release PDF/HTML/JSON evidence pack for the promoted v0.8/v0.9 numerical
  conformance cases, including accuracy and execution-time graphs.

Conformance claims included in this release are still limited to earlier
declared tolerance-gated cases:

- `heat_balance_nomass_001`: `Zone Mean Air Temperature`
- `surface_temperature_nomass_001`: zone MAT plus surface inside/outside face
  temperatures

## Not Included

v0.13.0 does not claim:

- plant numerical conformance
- plant loop flow balancing or operation-scheme parity
- pump head, pump power, boiler load, chiller load, or condenser-loop parity
- plant node temperature, humidity ratio, mass-flow, or setpoint parity
- plant meters or fuel accounting parity
- plant ExampleFiles compatibility
- HVAC, node, IdealLoads, sizing, autosizing, availability-manager, or general
  ExampleFiles compatibility beyond already promoted cases

The plant skeleton fixture remains:

```text
evidence_level: smoke
conformance_claim: false
tolerance_policy: none
status: typed graph only
```
