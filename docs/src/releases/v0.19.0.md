# v0.19.0 Release Notes

v0.19.0 closes the series reader and compare engine v2 milestone.

## Scope

- adds `SeriesComparisonV2` with index or timestamp alignment
- reports compared sample count, max absolute delta, RMSE, max relative delta,
  status, and first divergence reason
- adds `SeriesSample` for timestamp-aware selected output series
- adds timestamp-aware ESO selected-series parsing with dictionary metadata,
  units, frequency, and hourly timestamp labels
- adds `compare-series-v2-smoke` and `v0.19-verify`

## Release Artifacts

- `dist/eplus-rs-v0.19.0-windows-x64.zip`
- `numeric-conformance-evidence.html`
- `numeric-conformance-evidence.pdf`
- `numeric-conformance-evidence.json`

## Claim Boundary

This release does not add numerical conformance. It is comparison
infrastructure for future selected output and meter reports.

Promoted tolerance-gated numerical conformance remains limited to:

- `heat_balance_nomass_001`
- `surface_temperature_nomass_001`

Meter conformance remains out of scope for v0.19.

## Next

v0.20 should promote release report generation and coverage matrices on top of
the v0.19 comparison metric layer.
