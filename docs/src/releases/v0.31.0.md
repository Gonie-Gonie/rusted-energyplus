# v0.31.0 Release Notes

v0.31.0 adds the release evidence asset manifest. The manifest records the
release package and the generated evidence documents that should be uploaded
beside the GitHub Release notes.

This release does not add new numerical conformance. It adds release
documentation infrastructure so users can see which package, coverage,
evidence, hashes, content types, and report summaries belong to the release.

## Added

- `release-evidence-manifest` release command
- `release_evidence_manifest.py` oodocs PDF/HTML/JSON/Markdown generator
- GitHub Release asset manifest docs
- v0.31 milestone and release verification script

## Release Evidence

The generated manifest covers:

- `dist/eplus-rs-v0.31.0-windows-x64.zip`
- numeric conformance evidence PDF/HTML/JSON
- conformance index PDF/HTML/JSON/Markdown
- support coverage PDF/HTML/JSON/Markdown
- file size, SHA-256, content type, producer command, and user-facing purpose
