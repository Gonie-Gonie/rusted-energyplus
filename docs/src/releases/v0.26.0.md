# v0.26.0 Release Notes

v0.26.0 closes the Internal Convective Gains Conformance milestone.

This release promotes the already implemented OtherEquipment convective-gain
runtime trace from smoke evidence to a declared, tolerance-gated conformance
case with release evidence artifacts.

## Added

- `compare-internal-convective-gain-conformance`
- `conformance internal-gains-report`
- timestamp-aligned markdown and JSON comparison artifacts for
  `internal_gains_001`
- v0.26 release verification

## Changed

- `internal_gains_001` now has `comparison_class = "conformance"` and
  `conformance_claim = true`.
- only `Zone Total Internal Convective Heating Rate` is promoted to
  conformance level in that fixture.
- static OtherEquipment nominal EIO rows in the same fixture remain diagnostic.
- the numerical release evidence PDF/HTML/JSON now includes 5 promoted cases
  and 7 promoted series.
- the internal convective gain CLI path moved into
  `crates/ep_cli/src/internal_gains.rs` to reduce `main.rs` growth.

## Claim Boundary

The v0.26 claim is limited to:

- `internal_gains_001`
- `Zone Total Internal Convective Heating Rate`
- hourly EnergyPlus ESO samples
- declared `internal-gain` tolerance policy

It does not claim zone air temperature response to internal gains, radiant
internal-gain coupling, latent or moisture coupling, HVAC compatibility, plant
compatibility, meter conformance, general heat-balance compatibility, or broad
ExampleFiles compatibility.
