# v0.18.0 Release Notes

v0.18.0 closes the output request injection and official oracle baseline
pipeline milestone.

## Scope

- stages copied IDFs with manifest-owned `Output:Variable` and `Output:Meter`
  requests
- skips output requests already present in the source IDF so existing fixtures
  are not duplicated when staged
- writes expanded baseline manifests with output-injection metadata
- adds an official `1ZoneUncontrolled.idf` baseline-only case using the
  repo-local EnergyPlus 26.1.0 ExampleFiles tree
- adds `official-baseline-smoke` and `v0.18-verify`

## Release Artifacts

- `dist/eplus-rs-v0.18.0-windows-x64.zip`
- `numeric-conformance-evidence.html`
- `numeric-conformance-evidence.pdf`
- `numeric-conformance-evidence.json`

## Claim Boundary

This release does not add numerical conformance. The official ExampleFiles case
is baseline-only and keeps `conformance_claim = false`.

Promoted tolerance-gated numerical conformance remains limited to:

- `heat_balance_nomass_001`
- `surface_temperature_nomass_001`
