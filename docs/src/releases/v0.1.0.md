## Scope

Included:

- parse epJSON into RawModel
- preserve object type and object name hierarchy
- preserve raw field values
- preserve unknown object types
- extract `Version.version_identifier`
- summarize object counts
- report tracked/untracked seed coverage from CLI
- preview typed compile summary for seed fixtures and oracle-generated smoke epJSON
- build a release zip artifact

Not included:

- full schema validation
- complete typed model conversion
- complete name reference resolution
- graph validation
- simulation
