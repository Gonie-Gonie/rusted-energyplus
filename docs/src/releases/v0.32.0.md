# v0.32.0 Release Notes

v0.32.0 adds the user coverage handbook. The handbook is generated from the
support coverage and conformance index JSON files, then rendered as
PDF/HTML/JSON/Markdown with `oodocs` and matplotlib.

This release does not add new numerical conformance. It makes the current
supported input, output, algorithm, promoted-case, and known-gap scope easier
for users to read before running an IDF.

## Added

- `user-coverage-handbook` release command
- `user_coverage_handbook.py` oodocs/matplotlib generator
- mdBook page for the user coverage handbook
- user coverage handbook release assets in the release evidence manifest
- v0.32 milestone and release verification script

## Release Evidence

The generated handbook covers:

- typed input objects
- structural input objects
- conformance output variables
- diagnostic and baseline output variables
- limited-conformance algorithms
- diagnostic-only algorithm projections
- promoted conformance cases
- known gaps and decision rules
