# v0.20.0 Release Notes

v0.20.0 closes the conformance report generator milestone.

## Scope

- adds a Python `oodocs` conformance index report generator
- writes release-level case, output, meter, domain, report, and gate coverage
  matrices
- emits `conformance-index.md`,
  `conformance-index-report.html`,
  `conformance-index-report.pdf`, and
  `conformance-index-report.json`
- adds `conformance-index-report` and `v0.20-verify`
- packages the report wrapper and Python generator with the release zip

## Release Artifacts

- `dist/eplus-rs-v0.20.0-windows-x64.zip`
- `conformance-index.md`
- `conformance-index-report.html`
- `conformance-index-report.pdf`
- `conformance-index-report.json`
- `numeric-conformance-evidence.html`
- `numeric-conformance-evidence.pdf`
- `numeric-conformance-evidence.json`

## Claim Boundary

This release does not add numerical conformance. The conformance index maps
coverage and reporting contracts; it does not promote cases by itself.

Promoted tolerance-gated numerical conformance remains limited to:

- `heat_balance_nomass_001`
- `surface_temperature_nomass_001`
