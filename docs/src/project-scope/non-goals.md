# Non-goals

Current public scope does not claim:

- EnergyPlus heat-balance compatibility
- HVAC compatibility
- plant-loop compatibility
- zone-temperature conformance
- meter conformance
- full runtime simulation compatibility

Development-only commands can exist before these goals are met, but their docs
and output must say that they are not conformance evidence.
